void WiFi2ControlClass::setWiFiConfig ( WiFiMode_t  mode,
const char *  apName,
const char *  apPwd 
)

initialize WiFi as access point or access point plus station mode.

Parameters
modeThe WiFi mode. Can be WIFI_STA, WIFI_AP or WIFI_AP_STA.
apNameThe name that the board will show as access point in WIFI_AP_STA and WIFI_AP mode.
apPwdThe password that will be required to the ones that will try to connect to our access point.

Example usage:

#include "WiFi2Control.h"
#define MY_AP_SSID "MyAP"
#define MY_AP_PASSWORD "Password"
void setup() {
wifi2control.begin();
wifiLink.setWiFiConfig(WIFI_AP, MY_AP_SSID, MY_AP_PASSWORD);
}
void loop() {
wifi2control.poll();
}
