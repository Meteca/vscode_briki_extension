41 /*
42 WL_CONNECTED: assigned when connected to a WiFi network;
43 WL_NO_SHIELD: assigned when no WiFi shield is present;
44 WL_IDLE_STATUS: it is a temporary status assigned when WiFi.begin() is called and remains active until the number of attempts expires (resulting in WL_CONNECT_FAILED) or a connection is established (resulting in WL_CONNECTED);
45 WL_NO_SSID_AVAIL: assigned when no SSID are available;
46 WL_SCAN_COMPLETED: assigned when the scan networks is completed;
47 WL_CONNECT_FAILED: assigned when the connection fails for all the attempts;
48 WL_CONNECTION_LOST: assigned when the connection is lost;
49 WL_DISCONNECTED: assigned when disconnected from a network;
50 */
