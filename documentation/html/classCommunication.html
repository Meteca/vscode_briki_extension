◆ getGenericData()

uint32_t Communication::getGenericData ( uint8_t *  data = NULL,
uint32_t  len = 0 
)

Retrieve custom data received.

If an external buffer was previously specified with initExternalBuffer this function will just clear available data flag since data are already available in buffer.
Otherwise data will be stored in a dynamically created buffer and with this function a copy of the available data will be pasted in the pointer passed as first parameter.

Parameters
dataPointer where data read will be placed.
lenLength of the data to read.
Returns
number of bytes read.

Example usage:

uint8_t buff[100];
void setup() {
Serial.begin(115200);
}
void loop() {
uint32_t avail = communication.availGenericData();
if(avail){
communication.getGenericData(buff, avail);
Serial.write(buff, avail);
}
}

register a custom command in order to be able to send and receive custom messages from communication microcontroller.

Parameters
cmdCodeThe code associated to the custom command.
functionA callback that will be called every time the custom command so created is received.
bufferA container of the custom message sent from the control microcontroller.
Returns
true if operation succedeed, false otherwise.

Example usage:

uint8_t commandData[50];
void setup() {
Serial.begin(115200);
communication.registerCommand(0x01, myCallbackFunction, commandData);
}
void loop() {}
void myCallbackFunction(uint32_t commandLen){
Serial.println("Custom command received:");
Serial.write(commandData, commandLen);
}
